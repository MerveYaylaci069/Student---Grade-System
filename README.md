# Student---Grade-System
An automation system that stores student status in a database, developed using Python programming.
Bu proje, Python, Flask ve MySQL kullanarak hazırlanmış bir **mini öğrenci otomasyon sistemidir**.  
Öğrencilerin bilgileri girilir, vize ve final notları alınır, ortalama hesaplanır ve geçip geçmedikleri ekranda gösterilir.

## Özellikler
- Öğrenci bilgilerini (Ad, Soyad, Şifre) kaydetme
- Vize ve Final notları girme
- Ortalama hesaplama
- Geçti/Kaldı durumunu renkli gösterme
- MySQL veritabanına kaydetme
- Basit web arayüzü (Bootstrap kullanılmıştır)

  ## Teknolojiler
  -Python 3
  -Flask
  -MySQL
  -Bootstrap 5

  ## Calısma
  - Vs code terminaline şu komut yazılmalı : python app.py
  - Tarayıcıya girilecek adres : http://127.0.0.1:5000
